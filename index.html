  • html - how does it work
  • tags - to describe a website structure
  • p - paragraph
  • CSS - to change how my html looks
  • head - meta information
  • body - contents of the page
  • h1-h6 - headings from biggest to smallest font size to provide visual context
  • title - name of the page displayed on the top left
  • img - to display image on html
  • ul - creates unorder list
  • ol - creates ordered list
  • a - anchor displays browser as a link text
  • div - divides the html documents into sections
  • span - an inline unit for small chunks of html
  • table - wraps the whole table
  • tr - wraps content into the same row
  • th - header for the table
  • td - data for the table
  • iframe - specifies an inline frame
